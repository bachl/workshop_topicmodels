4.1 Test der Modellqualit√§t

  • {oolong} ist ein recht neues Paket, das die Durchf√ºhrung von standardisierten Tests der Topic-Qualit√§t erleichtert.   • Es werden zwei Tests angeboten:

    • Der topic intrusion test zeigt eine Auswahl von Dokumenten aus dem Korpus. Dazu werden eine vorgegebene Zahl von Topics angezeigt, die in einem Dokument am wahrscheinlichsten enthalten sind.     • Diese Tests sollten von mehreren Personen durchgef√ºhrt werden. Im Idealfall werden die Modelle auch durch Personen getestet, die nicht am Projekt (oder zumindest nicht an der Modellierung) beteiligt waren, um die intersubjektive Nachvollziehbarkeit der Interpretationen zu testen.
    • Aber schon formale Tests nur mit den Projektbeteiligten schlagen den heute √ºblichen Standard um Weiten. Bisher werden zur Validierung vor allem informelle Diskussionen eingesetzt (Maier et al. 2018) ‚Äî so auch in unserer Beispielstudie.
